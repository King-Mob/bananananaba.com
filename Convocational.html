Convocational Development

It Takes A Village To Make An App

What if an app was made democratically?

Convocational Development is a perspective on making software that treats the users and developers as forming a miniature society.

Most large software platforms are run like aristocratic kingdoms. The users have no formal power to change the platform itself: their influence is purely reactive.

I want to build software where users have a signficant say in the direction the development goes in, to the point where the line between a user and a developer starts to blur.

To do this requires a democratic apparatus, and the building of these apparatuses is what this project is about.
